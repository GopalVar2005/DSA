Problem Statement: Given an array of N integers, left rotate the array by one place.

Appr1:
use another array to store the numbers in the desired way.
TC-O(N)
SC-O(N)

Appr2:
        int n=arr.length;
        int temp=arr[n-1];
        for(int i=n-1;i>0;i--)
        {
            arr[i]=arr[i-1];
        }
        arr[0]=temp;
TC-O(N)
SC-O(1)
